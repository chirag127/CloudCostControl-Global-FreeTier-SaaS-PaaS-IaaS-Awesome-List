## üåü Free-Tier Cloud Services

### IaaS (Infrastructure as a Service)

*   **AWS Free Tier**: EC2 (750 hours/month), S3 (5GB Standard Storage), Lambda (1M free requests), etc. [Explore AWS Free Tier](https://aws.amazon.com/free/)
*   **Google Cloud Free Tier**: Compute Engine (f1-micro instance), Cloud Storage (5GB Standard), BigQuery (1TB queries/month), etc. [Explore Google Cloud Free Tier](https://cloud.google.com/free)
*   **Azure Free Account**: 12 months of popular free services, plus $200 credit. [Explore Azure Free Account](https://azure.microsoft.com/en-us/free/)
*   **Oracle Cloud Free Tier**: Always Free Autonomous Database, Compute VMs, Block Volume, Object Storage. [Explore Oracle Cloud Free Tier](https://www.oracle.com/cloud/free/)

### PaaS (Platform as a Service)

*   **Heroku Free Tier**: Limited dyno hours for small applications. (Note: Free tier often requires activity to remain active). [Explore Heroku Pricing](https://www.heroku.com/pricing)
*   **Vercel**: Free for personal projects, including serverless functions, global CDN. [Explore Vercel Pricing](https://vercel.com/pricing)
*   **Netlify**: Free for personal sites, including CI/CD, custom domains, SSL. [Explore Netlify Pricing](https://www.netlify.com/pricing/)
*   **Render**: Free tiers for web services, databases, static sites. [Explore Render Pricing](https://render.com/pricing)

### SaaS (Software as a Service)

*   **GitHub**: Free for public and private repositories, includes Actions minutes, Packages storage. [Explore GitHub Plans](https://github.com/pricing)
*   **GitLab**: Free for public and private repositories, includes CI/CD minutes. [Explore GitLab Plans](https://about.gitlab.com/pricing/)
*   **Figma**: Free for individuals, includes limited projects and collaborators. [Explore Figma Pricing](https://www.figma.com/pricing/)
*   **Notion**: Free for personal use, unlimited pages & blocks. [Explore Notion Pricing](https://www.notion.so/pricing)

### Databases & Storage

*   **MongoDB Atlas Free Tier**: M0 Sandbox clusters (512MB storage). [Explore MongoDB Atlas](https://www.mongodb.com/cloud/atlas/free-tier)
*   **Supabase Free Tier**: Postgres database, authentication, storage for small projects. [Explore Supabase Pricing](https://supabase.com/pricing)
*   **PlanetScale Free Tier**: Generous free tier for a serverless MySQL database. [Explore PlanetScale Pricing](https://planetscale.com/pricing)
*   **Cloudflare R2 Storage**: Free up to 10 GB/month. [Explore Cloudflare R2](https://developers.cloudflare.com/r2/pricing/)

### AI/ML & Serverless

*   **Google Colab**: Free GPU/TPU access for machine learning research and development. [Explore Google Colab](https://colab.research.google.com/)
*   **Hugging Face Free Tier**: Access to thousands of pre-trained models, limited inference API calls. [Explore Hugging Face Pricing](https://huggingface.co/pricing)
*   **OpenAI Free Tier**: Initial free credits for API usage. (Note: Subject to change, check current offers). [Explore OpenAI Pricing](https://openai.com/api/pricing/)

### Monitoring & Analytics

*   **Grafana Cloud Free Tier**: 10k series, 50GB logs, 50GB traces. [Explore Grafana Cloud](https://grafana.com/products/cloud/free/)
*   **Datadog Free Tier**: 5 host metrics, 100 log events/month, 100 traces/month. [Explore Datadog Pricing](https://www.datadoghq.com/pricing/)
*   **Sentry Free Tier**: 5k error events/month, 10k transaction events/month. [Explore Sentry Pricing](https://sentry.io/pricing/)

### Security & Identity

*   **Auth0 Free Tier**: Up to 7,000 active users, unlimited logins. [Explore Auth0 Pricing](https://auth0.com/pricing)
*   **Clerk.dev Free Tier**: Up to 5,000 monthly active users. [Explore Clerk.dev Pricing](https://clerk.com/pricing)
*   **Cloudflare Free**: Basic DDoS protection, CDN, DNS. [Explore Cloudflare Plans](https://www.cloudflare.com/plans/)

### Developer Tools & CI/CD

*   **Travis CI Free Tier**: Limited build minutes for open-source projects. [Explore Travis CI Pricing](https://www.travis-ci.com/pricing)
*   **CircleCI Free Tier**: Limited credits for build minutes. [Explore CircleCI Pricing](https://circleci.com/pricing/)
*   **Postman Free Tier**: Limited collections, requests, and team members. [Explore Postman Pricing](https://www.postman.com/pricing/)
